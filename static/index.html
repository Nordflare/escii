1.1 - Purpose of creation and use

  1. Encourage usage of characters from European languages and European way of living.
  2. Enable ease of adding more Europe-specific characters in the future when needed. For example if a euro-cent symbol ever gets introduced using ESCII wouldn't require needing to apply for the symbol to be used in another charset such as UTF-8 or ASCII.
  3. Promote European standards for European internet. ASCII is an acronym for American Standard Code for Information Interchange. European internet shouldn't have to rely on standards that can only have been intended as only for America or only for a certain country.
  4. Those who speak a certain language know best how to interperate that language's characters in computer format. Therefore that power should be given to those who speak the languages in question. By having a European standard charset that power transfer would be facilitated.

1.3 - Notice

The above table supports latin characters only, cyrillic, greek and other characters are supported in their respective sub-dividions of ESCII as ESCII:Cyrillic, ESCII:Greek e.t.c. If you only need latin characters and want no symbols use ESCII:Latin. Symbols are found in ESCII:Symbols and currencies are in ESCII:Currency.
